1. Grouping selectors

x, y {...}

e.g.

article, p {...} or .group1, .group2 {...}

Selectors separated by a comma

Rules will be applied to both selectors.

this is a para w/ class of group3

2. Descendant combinator

x y {...}

e.g.

article p {...} or article .class {...}

Selectors separated by space

Read from right to left.

In the first example rules will be applied to every p (paragraph) element that is a descendant of an article element.

In the second example rules will be applied to every element that has a class attribute value of class that is a descendant of an article element.

üìù Descendant combinators in action ...

para 1 in section

para 2 in article (nested)

para 3 in article with class

para 4 in article with a span class

I'm an article with a class

3. Direct child combinator

x > y {...}

e.g.

ul > li {...} or ul > .li_class {...}

Selectors separated by a greater than sign.

Read from right to left.

In the first example rules will be applied to every li element that is a direct child of a ul element.

In the second example rules will be applied to every element that has a class attribute value of li_class that is a direct child of a ul element.

üìù Direct child combinator in action...

  • ul list item 1
    1. ol list item 1
    2. ol list item 2
    • this is a ul list item w/ class li_class
  • ul list item 2

4. General sibling combinator

x ~ y {...}

e.g.

q ~ samp {...}

Selectors separated by a tilde

Read from right to left.

In the example the rule applied to every samp (sample computer output) element (could be multiple elements) that is placed after a q (quote) element.

üìù General sibling combinator in action...

quote

paragraph 1

Error msg 1 | Error msg 2

paragraph 2

Error msg 3

Adjacent sibling combinator

x + y {...}

e.g.

q + small {...}

Two selectors separated by a plus sign

Read from right to left.

In the example this is the only small element that comes right after an q (quote) element.

üìù Adjacent sibling combinator in action...

quote small text 1 | small text 2
